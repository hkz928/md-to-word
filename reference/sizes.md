# Chinese Character Size Reference

## Common sizes for official documents

| 中文称号 | 磅值 | 毫米 | Usage |
|---------|------|------|-------|
| 三号 | 16pt | ~5.6mm | Headings, body |
| 小三 | 15pt | ~5.3mm | Subheadings |
| 四号 | 14pt | ~4.9mm | Notes |
| 小四 | 12pt | ~4.2mm | Regular text |

## Line spacing conventions

| Context | Line spacing |
|---------|--------------|
| Headings | 30pt |
| Body | 28pt |
| Notes | 24-26pt |

## Spacing conversions

```
1 cm ≈ 28.35 points
2 cm ≈ 56.7 points
0.5 line ≈ 12 points (for 16pt font)
```

## Margin standards

| Document type | Margins |
|---------------|---------|
| Official documents | 2cm all sides |
| Regular documents | 2.54cm (1 inch) all sides |
| Narrow | 1.27cm (0.5 inch) all sides |
